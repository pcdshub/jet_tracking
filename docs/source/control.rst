Jet Control Overview
####################

To start IPython for CXI hutch-python3:

.. code-block:: bash

    /reg/g/pcds/pyps/apps/hutch-python/cxi/cxipython

There is a soft link in ``~cxiopr/bin``, so that on CXI machines, one can start
IPython as follows:

.. code-block:: bash

    cxi3
